# ED00 Labeler

Manual milk labeler designed by EDMS ([web site](https://www.ed-ms.fr))

## 3D printed parts

### 3D printing recommended parameters
- Material : PETG
- Layers height : 0.3mm
- Number of perimeters : 2 mini
- Part filling : 20%
- No suppport (except on 001001)
